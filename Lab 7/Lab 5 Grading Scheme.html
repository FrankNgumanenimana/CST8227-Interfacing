The learning objectives of the lab include using a capacitive (cap) touch sensor as an input device to control a Piezo buzzer and a LED.

To help you with the lab, I've published this rubric along with resources and release notes so that you can build your own projects :)

Level C: Piezo Knock Sensor

For a C grade, provide a working demo of a Piezo knock sensor.

C- if onboard LED (pin 13).

Implementation Notes

To make your own Piezo sounder alarm, see "Project 13 - Piezo Knock Sensor" in the eBook Beginning Arduino.

Notes:

  • Parts List :: no need for a 2-way screw terminal
  • Source Code :: notice the Piezo buzzer is connected to pin 5 (Listing 4-3). But of course that's for an Arduino. You'll need to select a different pin on the Teensy. Make sure the pin you select is an analog pin. Hint: pin 5 on the Teensy is not an analog pin (!).

You'll need to experiment with the threshold value to 'knock' the Piezo buzzer and turn on the LED. I printed the sensorValue to the Serial monitor. Once I saw the values that were being read from piezoPin, I adjusted the threshold value to be a little greater than the max(sensorValue) that I saw. In other words, I determined the threshold value empirically; I did it by experimentation, as opposed to scientifically, such as a formula.

Level A+: Capacitance Sensor

For an A+ grade, provide a working demo of a capacitance (cap) sensor as an input device to control a Piezo buzzer and a LED.

You'll need to perform independent research on how to make a cap sensor. The good news: it's relatively easy to make a cap sensor, and you can make one out of a piece of card-board and aluminum foil. A good analogy: like a key for a piano, except it's made out of aluminum foil.
